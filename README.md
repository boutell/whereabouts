whereabouts
==========

A status app for co-workers. 


To get this running

1. hop into the config directory and make a copy of settings.sample called settings.js.
2. open settings.js and add fill out the exports.people object with the people in your company/organization
3. in the root directory of the repo: $ node app.js
4. open a browser and go to localhost:3000/punks/generate -- DO THIS ONLY ONCE!!
5. navigate to locahost:3000 and you'll be good to go
